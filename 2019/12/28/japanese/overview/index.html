五十音/Gojūon

Many people says that the only way to learn Hiragana and Katagana is through rote learning. I didn’t know that when I started learning so I learnt with assistance of several apps :P.

  • 50音起源 is an app to learn Hiragana and Katakana by etymology.
  • Duolingo: Many people have used this app. I have been using this app to learn German before.

Particles

Wikipedia

Being a particle:

  • Topic marker は(ha) is pronounced as わ(wa)
  • へ(he) is pronounced as え(e)
  • を(wo) is pronounced as お(o)

Grammar

OBJECT → * | そ(so)う(u) (这样) | 初め(me) (初次)
WOMAN → OBJECT | OBJECT ち(chi)ゃ(ya)ん(n)
MAN → OBJECT | OBJECT さ(sa)ん(n)
PRONOUN → こ(ko)れ(re)(this) | そ(so)れ(re)((near) that) | あ(a)れ(re)((far) that) | 誰(who) | 何(what) | ど(do)れ(re)(which) | ど(do)じ(ji)る(ru)(which one of two)
THING → OBJECT | OBJECT の(no) (OBJECT)
NOUN → WOMAN | MAN | THING | PRONOUN

Declarative

  • DECLARATIVE →
    • (は(ha)い(i)、)(NOUN は(ha)) NOUN で(de)す(su)。 |
      A is B.
    • (い(i)い(i)え(e)、)(NOUN で(de)は(ha)) NOUN あ(a)り(ri)ま(ma)せ(se)ん(n)。 |
      A is not B.
    • 違(ち(chi)が(ga))い(i)ま(ma)す(su)。 That is remotely different.

Note: は(ha) is pronounced as わ(wa) in speaking. See particles.

Interrogative

INTERROGATIVE → DECLARATIVE か(ka)。

Is … true?

The か(ka) can be omitted in speaking. In that case a rising tone at the end is needed.
